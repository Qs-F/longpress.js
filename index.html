longpress.js

Press Here For a second!

Installation

npm install https://github.com/Qs-F/longpress.js.git

Demo

new LongPress(document.getElementsByTagName("button")[0], 1000, function(){})
new LongPress(document.getElementsByTagName("button")[1], 1000, function(){alert("you gain long press!")})

Or, please tap header once.

Usage

One line code

new LongPress(ELEMENT, INTERVAL, CALLBACK_FUNCTION)

General code

var l = new LongPress
l.SetElement(ELEMENT)
l.SetInterval(INTERVAL)
l.SetCallback(CALLBACK_FUNCTION)
